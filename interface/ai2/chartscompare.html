Charts Compare

Charts are based on your stations
readings. All charts are zoomable

The Select-a-chart allows you to configure your own graph to compare values together. Up to six different data values can be selected using the drop-down selectors below the graph. Each data value has a further drop-down selector to change the colour for that particular data plot. You can edit the colours in the file 'lib/highcharts/ax-theme.js'.
