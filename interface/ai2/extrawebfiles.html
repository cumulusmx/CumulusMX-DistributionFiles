Setup Extra Web Files

Use this page to add files to be uploaded / copied for your systems

Note: that you can use replacable values in both the source and target file fields for Cumulus files that change their name periodically. The allowed values are: <currentlogfile> <currentextralogfile> <airlinklogfile> <noaayearfile> <noaamonthfile> <custinterval[1-10]>
The tagged value will be replaced by the current logs file name.

Please note that the destination root is tied to the FTP user and may not be the root of your website.
