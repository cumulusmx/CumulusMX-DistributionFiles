AI Configuration

Use this page to configure the Alternative Interface

Page Geometry

Configure the margins used between the header, page & footer, the units used and whether or not the header & footer are static.

You should note that the footer will scroll automatically on small screens regardless of the above settings.

Theme

This version of the AI comes with 28 built-in themes. Use the dropdown to select - this page will instantly change to show your choice.

This facility only works on browsers that support 'local storage'. If you are using an old browser to view your Interface then please consider updating if possible. (You can also use the 'CMXConfi' settings below to manually change your theme.)

The Seagull

The seagull at the bottom right of the page can be animated in different ways. By setting to 'No animation' the seagull will just appear in its current position.

In order to see changes dependant on numeric values, you need to click elsewhere on the page.
